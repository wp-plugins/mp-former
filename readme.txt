Easy to use graphic UI for custom post type building. 

== Description ==

MP Former allows to create and manage the list of custom post types. MP Former 
allows to do next things:
 * Create unlimited list of custom post types;
 * Activate and deactivate already created;
 * Create custom list of taxonomies related to custom post type;
 * Create custom post statuses;
 * Manage list of columns (aka Show on Screen columns)

With MP Former you can :
  * Create a new custom post type and setup most important post options;
  * Add extra post statuses to custom post type;
  * Create list of custom taxonomies, related to post type;
  * Activate or deactivate existing custom post types;

== Installation ==

1. Upload `mp_former` forlder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
